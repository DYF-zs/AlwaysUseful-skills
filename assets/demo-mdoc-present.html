mdoc-present

Markdown → HTML 自包含 Glassmorphism Mermaid

mdoc-present 是一个 AI Skill，用于将任意 Markdown 文档转换为自包含的规范化 HTML 演示页。输出风格统一使用 warm paper 背景、玻璃态卡片、编辑式排版和语义化徽章系统。

01

核心 Workflow

  1. 完整读取源 Markdown
  2. 若本地存在规范模板，读取并校准视觉决策
  3. 读取 design-tokens.md 获取样式契约
  4. 若含 Mermaid 代码块，读取 mermaid-setup.md
  5. 从首个 # 标题推导文档标题
  6. 生成单个自包含 .html 文件
  7. 预览并验证输出质量
02

不可协商的样式规则

Warm Paper

SVG noise + #faf9f6 → #f0ece4 垂直渐变背景

字体栈

Noto Serif SC 标题 / Noto Sans SC 正文 / JetBrains Mono 代码

Glass 卡片

rgba(255,255,255,.45) 背景 + backdrop-filter blur(24px)

03

Markdown 映射规则

Markdown 输入 HTML 输出 MVP

## Heading <section class="section reveal"> + 编号 Yes
### Heading <div class="subsection"> Yes
Markdown 表格 .table-wrap > table Yes
```mermaid .flowchart-wrap > pre.mermaid Partial
Blockquote .note 或 .note-block No
